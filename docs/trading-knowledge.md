# A股短线技术分析知识库（供预测引擎参考）

> 本文档供 AI 预测模块与开发者参考，汇总 A 股短线交易常用技术指标的计算公式、
> 信号判定规则与量价关系经验，作为 `app/indicators.py` 实现与 GLM 预测提示词的依据。
> 来源：综合网络公开的技术分析资料（见文末）。

## 一、移动平均线 MA

- **计算**：MA(N) = 最近 N 日收盘价的算术平均。常用 MA5 / MA10 / MA20 / MA60。
- **多头排列**：MA5 > MA10 > MA20 > MA60 且同步向上 → 趋势强势，短线偏多。
- **空头排列**：MA5 < MA10 < MA20 < MA60 → 趋势走弱，短线偏空。
- **金叉**：短期均线上穿长期均线（如 MA5 上穿 MA10）→ 买入信号。
- **死叉**：短期均线下穿长期均线 → 卖出信号。
- **支撑/压力**：上方均线常构成压力，下方均线常构成支撑；股价站上 MA20 视为中期转强。

## 二、MACD（指数平滑异同移动平均线）

- **计算**：
  - EMA12、EMA26（12/26 日指数移动平均）
  - DIF = EMA12 − EMA26（快线）
  - DEA = DIF 的 9 日 EMA（慢线/信号线）
  - MACD 柱 = 2 ×（DIF − DEA）
- **信号**：
  - **金叉**：DIF 上穿 DEA（柱由绿转红）→ 买入。
  - **死叉**：DIF 下穿 DEA（柱由红转绿）→ 卖出。
  - **零轴上方金叉**比零轴下方金叉更可靠（多头趋势中）。
  - **柱状放大**：红柱拉长 = 上涨动能增强；绿柱拉长 = 下跌动能增强。
  - **顶/底背离**：股价创新高而 DIF 未创新高 → 顶背离，警惕回落；反之为底背离。

## 三、KDJ（随机指标）

- **计算**（9 日）：
  - RSV =（收盘 − 近9日最低）/（近9日最高 − 近9日最低）× 100
  - K = 2/3 × 前K + 1/3 × RSV（初值 50）
  - D = 2/3 × 前D + 1/3 × K（初值 50）
  - J = 3K − 2D
- **信号**：
  - **超买**：K > 80 或 J > 100 → 短期偏高，回调风险。
  - **超卖**：K < 20 或 J < 0 → 短期偏低，反弹机会。
  - **金叉**：K 上穿 D（尤其在低位 20 以下）→ 买入。
  - **死叉**：K 下穿 D（尤其在高位 80 以上）→ 卖出。
  - KDJ 灵敏但震荡市易钝化/失真，需与趋势指标配合。

## 四、RSI（相对强弱指标）

- **计算**（通达信式，N 日）：
  - RSI = 100 × SMMA(上涨幅度, N) / SMMA(绝对涨跌幅, N)
  - 常用 RSI6 / RSI12 / RSI24。
- **信号**：
  - **超买**：RSI > 70（强势市 80）→ 谨慎追高。
  - **超卖**：RSI < 30（弱势市 20）→ 修复需求。
  - RSI6 上穿 RSI12 → 短线转强；下穿 → 转弱。
  - 强趋势中 RSI 可长期钝化于高位/低位，不可单独作为反转依据。

## 五、布林带 BOLL（20 日，2 倍标准差）

- **计算**：
  - 中轨 MID = MA20
  - 上轨 UP = MID + 2 × STD20（20 日收盘价总体标准差）
  - 下轨 LOW = MID − 2 × STD20
- **信号**：
  - 价格触/破**上轨**：强势但有超买，注意是否滞涨。
  - 价格触/破**下轨**：弱势但有超卖，注意是否企稳。
  - **开口**（带宽放大）→ 趋势行情启动；**收口**（带宽收窄）→ 变盘临近。
  - 三轨同向上 = 强势续涨；同向下 = 弱势续跌。
  - 股价回踩中轨不破常为上涨中继的买点。

## 六、量价关系（核心经验）

理想形态：**价涨量增、价跌量缩**。出现背离要警惕。

| 量价模式 | 含义 | 操作倾向 |
|----------|------|----------|
| 量增价涨 | 资金共识、动能充足 | 健康上涨，偏多 |
| 低位放量突破 | 主力进场/试盘，趋势反转 | 关注买点 |
| 底部缩量企稳 | 抛压衰竭 | 等温和放量买入 |
| 量缩价涨 | 惜售但接力不足（背离） | 警惕滞涨 |
| 量增价跌 | 抛压沉重（背离） | 偏空，回避 |
| 高位放量滞涨 | 主力出货，多空分歧 | 减仓/卖出 |
| 高位放量下跌 | 资金出逃 | 偏空，止损 |
| 缩量下跌 | 阴跌、无承接 | 观望等缩量企稳 |

- **放量**：成交量 ≥ 近 5 日均量的 1.5～2 倍（换手率显著抬升）。
- **缩量**：成交量 ≤ 近 5 日均量的 0.7 倍。

## 七、指标组合（提升胜率）

单一指标易误判，多指标同向共振才可靠：

- **趋势确认组合 MA + MACD**：均线多头排列 + MACD 零轴上金叉 → 高质量多头。
- **极值捕捉组合 BOLL + RSI**：价格触下轨 + RSI 超卖 → 反弹买点；触上轨 + RSI 超买 → 滞涨卖点。
- **高胜率共振买点**：KDJ 低位金叉 → MACD 金叉柱翻红 → 股价站稳 BOLL 中轨 → 多指标共振。
- **大盘环境**：个股需顺大盘（上证/创业板）方向，逆势强行做多胜率低。

## 八、应用到次日涨跌预测的判定逻辑

综合打分（偏多 +1 / 偏空 −1）：
1. 均线多头排列且站上 MA20 → +1；空头排列 → −1
2. MACD 金叉/红柱放大 → +1；死叉/绿柱放大 → −1
3. KDJ 低位金叉 → +1；高位死叉/超买 → −1
4. RSI 超卖回升 → +1；超买 → −1
5. 价格触 BOLL 下轨 → +1（超跌反弹）；触上轨滞涨 → −1
6. 量价：量增价涨 → +1；高位放量滞涨/量增价跌 → −1
7. 顺大盘方向 → 强化，逆大盘 → 弱化

得分明显为正 → 预测涨；明显为负 → 预测跌；接近 0 → 平/震荡。
技术面给出方向与强度，再结合联网搜到的消息面（利好/利空/公告/政策）修正。
